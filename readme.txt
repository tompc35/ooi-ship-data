Shipboard water sampling data from OOI Endurance Array cruises. Posted on Github for use in Google Colab - Data Labs REU project 2020.

Obtained from https://alfresco.oceanobservatories.org

```Coastal Endurance Array > Cruise Data```

For each cruise, water sampling data can be found by navigating to:

```Ship Data > Water Sampling```

Discrete Summary spreadsheets and readme files included here.
